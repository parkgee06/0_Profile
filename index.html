  • 제작전략 - Coding Production strategy
    기여도 100%

    바닐라 스크립트와 다양한 Javascript 라이브러리, 플러그인, 그리고 AJAX 기법을 사용하여 복잡한 인터랙션과 동적인 기능을 구현했습니다. 특히, jQuery를 사용해 메인 비주얼 갤러리를 개발하고, 다양한 탭메뉴 등의 기능을 통해 사용자 경험을 향상시켰습니다.

    HTML5 / CSS3 / JavaScript / jQuery / AJAX / 그누보드(게시판) / W3C 웹표준 통합 마크업 검사(html, css) / 웹접근성검사(K-WAH) / 크로스브라우징 완료

  • 디자인개선 - Design System
    기여도 100%

    브랜드 색상과 가독성을 개선한 최신 UI 디자인을 적용하고, 사용자 중심의 직관적 레이아웃으로 네비게이션을 향상시켰습니다. 또한 AJAX와 스크립트를 활용하여 풍부한 정보 제공과 상호작용 기능을 추가해 사이트의 유용성을 높였습니다.

    Adobe Photoshop / illustrator / Figma
